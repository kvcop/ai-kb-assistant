# MCP (server-memory)

Эта директория зарезервирована под локальное состояние MCP‑серверов (например, `server-memory`).

По умолчанию всё содержимое `.mcp/` **игнорируется git** (см. `.gitignore`), чтобы случайно не закоммитить приватные данные.

Рекомендация:
- хранить файл памяти локально (например, `.mcp/server-memory.jsonl`);
- подключать его через `MEMORY_FILE_PATH` при запуске Codex / MCP.
