# Scripts

Утилиты для обслуживания базы знаний и интеграций.

- `kb.py` — основной entrypoint для ежедневного контура (бриф/итоги/вопросы/тайм‑репорт).
- `tmux_tools.py` — вспомогательные утилиты для работы с tmux (опционально).
- `speech2text.py` и `mcp_speech2text.py` — опциональная интеграция с сервисом speech-to-text (если используете голосовые).
- `mcp_telegram_send.py` и `mcp_telegram_followups.py` — MCP‑инструменты для отправки/получения follow-ups в Telegram.
