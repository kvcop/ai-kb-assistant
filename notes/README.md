# Notes

Каталог для базы знаний (Markdown).

Рекомендуемый минимальный контур:

- `notes/work/` — активная рабочая поверхность (ежедневные файлы + список задач/напоминаний).
- `notes/meetings/` — заметки и протоколы встреч.
- `notes/technical/` — технические заметки.
- `notes/ideas/` — идеи/черновики.
- `notes/daily-logs/` — исторические копии end-of-day (генерируется).
- `notes/personal/` — личные заметки (опционально; по умолчанию вне рабочего контура).
- `notes/rostock/` — отдельный контур (опционально; если нужно отделять сайд‑проекты).
