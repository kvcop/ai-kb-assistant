# Work notes

Рабочий ежедневный контур.

Скрипт `scripts/kb.py` генерирует/обновляет:
- `notes/work/daily-brief.md` — утренний бриф.
- `notes/work/end-of-day.md` — контейнер дня (заметки + черновик списания времени).
- `notes/daily-logs/YYYY-MM-DD.md` — архивные копии end-of-day.

Файлы ниже — «долгоживущие» и обычно правятся редко:
- `jira.md` — канонический JQL (опционально).
- `reminders.md` — напоминания (используется ботом и `kb.py reminders`).
- `time-buckets.md` — список «корзин времени» (опционально).
- `todos.md` — ручной TODO‑бэклог по проектам.
- `open-questions.md` — отложенные вопросы (опционально).
- `typos.md` — глоссарий опечаток/терминов для транскрипций (опционально).
